About me

This is me

After a couple coding classes I started to lean more towards the development side. There is a lot of creativity in designing the way people interact with things.

I was thrilled with the ability to not only design websites, but manipulate things based off what I coded.

So that's where I've been. Trying to learn many different web technologies and the techniques that accompany them to become a better programmer.
